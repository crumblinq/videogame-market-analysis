# Определение закономерностей, влияющих на коммерческий успех видеоигр

## О проекте
Исследовательский анализ исторических данных интернет-магазина "Стримчик", который продает компьютерные игры по всему миру.

## Цель проекта 
Выявить закономерности, определяющие успешность игры, чтобы помочь в планировании рекламных кампаний на 2017 год.

## Задача
На основе данных о продажах игр по регионам, пользовательских оценках и оценках критиков, жанрах, платформах, возрастных рейтингах ESRB нужно определить:
- какие платформы перспективны для продвижения,
- какие жанры наиболее прибыльны,
- как различаются предпочтения пользователей по регионам,
- связаны ли отзывы и продажи,
- подтверждаются ли статистические гипотезы.



## Что было сделано
- проведена предобработка данных:
  - приведены названия столбцов к нижнему регистру,
  - преобразованы типы данных,
  - обработаны пропуски,
  - проверены дубликаты,
  - рассчитаны суммарные мировые продажи;
- выполнен исследовательский анализ данных;
- определён актуальный период для прогноза на 2017 год;
- проанализированы продажи по платформам и жанрам;
- составлен портрет пользователя по регионам `NA`, `EU`, `JP`;
- проверены статистические гипотезы.

## Основные выводы
- для прогноза на 2017 год был выбран актуальный период `2013–2016`;
- наиболее перспективными платформами для рекламных кампаний выглядят `PS4` и `XOne`;
- `3DS` можно рассматривать как дополнительную платформу;
- оценки критиков связаны с продажами заметно сильнее, чем пользовательские оценки;
- в регионах `NA`, `EU` и `JP` предпочтения пользователей в платформах и жанрах различаются;
- японский рынок отличается более высоким интересом к портативным платформам.

## Результат
Проект позволяет определить наиболее перспективные платформы и жанры для продвижения игр, а также учитывать региональные различия в спросе при планировании маркетинговых кампаний.

## Описание данных
- `Name` — название игры
- `Platform` — платформа
- `Year_of_Release` — год выпуска
- `Genre` — жанр игры
- `NA_sales` — продажи в Северной Америке (миллионы проданных копий)
- `EU_sales` — продажи в Европе (миллионы проданных копий)
- `JP_sales` — продажи в Японии (миллионы проданных копий)
- `Other_sales` — продажи в других странах (миллионы проданных копий)
- `Critic_Score` — оценка критиков (максимум 100)
- `User_Score` — оценка пользователей (максимум 10)
- `Rating` — рейтинг от организации ESRB (англ. Entertainment Software Rating Board). Эта ассоциация определяет рейтинг компьютерных игр и присваивает им подходящую возрастную категорию.

Данные за 2016 год могут быть неполными.
